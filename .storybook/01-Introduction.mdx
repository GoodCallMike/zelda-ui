# Zelda UI

A modern React component library built with TypeScript, featuring accessible components with beautiful design.

## ðŸš€ Quick Start

### Installation

Install the packages you need:

```bash
# Core components and theme (required)
npm install @zelda/core @zelda/theme

# Icons (optional)
npm install @zelda/icons
```

### Basic Usage

```tsx
import { Button, Input } from '@zelda/core';
import { SearchLgIcon } from '@zelda/icons';
import '@zelda/theme/dist/theme.css';

function App() {
  return (
    <div>
      <Button variant="primary" icon={SearchLgIcon}>
        Search
      </Button>
      <Input placeholder="Enter your quest..." />
    </div>
  );
}
```

## ðŸŽ¨ Design System

Zelda UI uses a cohesive design system with:

- **Primary colors** that adapt automatically in dark mode
- **Secondary colors** for success states
- **Tertiary colors** for information
- **Destructive colors** for errors
- Premium typography with carefully selected font families
- Consistent spacing, shadows, and border radius scales

## â™¿ Accessibility

All components are built with accessibility as a core principle:

- WCAG 2.1 AA compliance
- Keyboard navigation support
- Screen reader compatibility
- Focus management
- High contrast support

## ðŸŒ™ Dark Mode

Dark mode is supported out of the box. Add the `dark` class to your root element:

```tsx
<html className="dark">
  {/* Your app */}
</html>
```

## ðŸ§ª Testing

Zelda UI includes comprehensive testing utilities:

```tsx
import { quickA11yTest, runAccessibilityTestSuite } from '@zelda/core/utils';

// Quick accessibility test
await quickA11yTest(<YourComponent />);

// Comprehensive testing
await runAccessibilityTestSuite(<YourComponent />, {
  expectedFocusableElements: 2,
  ariaAttributes: { 'button': 'Submit form' },
});
```

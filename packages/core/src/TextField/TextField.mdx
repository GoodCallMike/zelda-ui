# TextField

A fully accessible text input component with label, description, and error message support.

## Features

- **Native HTML Input**: Built on standard input element for maximum compatibility
- **Accessibility Built-in**: Proper labeling, ARIA attributes, and screen reader support
- **Validation States**: Visual and semantic error handling
- **Flexible Styling**: Supports all native input attributes and custom styling

## Best Practices

- **Always provide labels** for screen readers and usability
- **Use descriptions** to provide helpful context or formatting requirements
- **Show errors clearly** with both visual and text indicators
- **Mark required fields** with both visual (*) and semantic indicators
- **Use appropriate input types** (email, password, tel, etc.)

## Accessibility

- ✅ **Proper Labeling**: Labels are correctly associated with inputs
- ✅ **ARIA Attributes**: `aria-describedby` and `aria-invalid` for context
- ✅ **Error Handling**: Errors are announced to screen readers
- ✅ **Keyboard Navigation**: Standard Tab and input behavior
- ✅ **Required Fields**: Semantic `required` attribute

## Examples

```tsx
// Basic usage
<TextField 
  label="Email Address"
  placeholder="Enter your email"
  onChange={(value) => setEmail(value)}
/>

// With validation
<TextField 
  label="Password"
  type="password"
  required
  errorMessage={passwordError}
  description="Must be at least 8 characters"
/>

// Controlled component
<TextField 
  label="Username"
  value={username}
  onChange={setUsername}
  description="Only letters, numbers, and underscores"
/>

// With custom styling
<TextField 
  label="Search"
  placeholder="Search products..."
  className="w-full max-w-md"
/>
```

## Form Integration

```tsx
// With form libraries
<form onSubmit={handleSubmit}>
  <TextField 
    label="Full Name"
    name="fullName"
    required
    onChange={(value) => setFieldValue('fullName', value)}
    errorMessage={errors.fullName}
  />
  
  <TextField 
    label="Email"
    type="email"
    name="email"
    required
    onChange={(value) => setFieldValue('email', value)}
    errorMessage={errors.email}
  />
</form>
```
